# Data Structures and Algorithms Practice

Simple Java Maven project for DSA practice.

## Structure

```
src/main/java/com/dsa/
├── Main.java
├── arrays/
├── heaps/
├── stacks/
├── trees/
└── slidingwindow/
```

## Quick Start

**Compile:**
```bash
mvn clean compile
```

**Run:**
```bash
java -cp target/classes com.dsa.Main
```

## Usage

Add your DSA problem solutions to the appropriate package under `src/main/java/com/dsa/`.
